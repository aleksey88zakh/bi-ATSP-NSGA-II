//===============================================================================================//
Входные параметры алгоритма GA:
//===============================================================================================//
Останов алгоритма: \ITER или \TIME
\n - кол-во городов
	по умолчанию: 12

\N - размер популяции
	по умолчанию: 50

\ITER - кол-во итераций
	по умолчанию: 200

\RUNS - кол-во запусков
	по умолчанию: 1

\INPUT - имя входного файла (имя указывать в кавычках)
	по умолчанию: "Example_MTSP_m2_n20_N50_12.txt"

\PROBLEM_NAME - имя задачи из входного файла, которую надо решить (имя указывать в кавычках)
	по умолчанию: (решаются все задачи из файла)

\FILE_PARETO - имя файла, в котором содержиться мн-во Парето (имя указывать в кавычках)
	по умолчанию: (отсутствует, сравнение не проводится)

\ALPHA_LS - параметры alpha ЛП в ГА (задаются подряд через пробел)
	по умолчанию: 0.5 0.5 0.5

\BETA_LS - параметры beta ЛП в ГА (задаются подряд через пробел)
	по умолчанию: 0.5 0.5 0.5
	
\NO_LS_begin - без ЛП в начале
	по умолчанию: ЛП в начале работает
	
\NO_LS_end - без ЛП в конце
	по умолчанию: ЛП в конце работает	

\TIME - время работы алгоритма (сек.); указывается через пробел время работы над каждой задачей
		если какое-то время равно нулю или не указано, то алгоритм работает по числу итераций
		Пример: \TIME 23.5 0 56
		1-ая задачи выполняется 23.5 сек., 3-ья - 56 сек., 2-ая, 4-ая и далее - по числу итераций
	по умолчанию: работа по числу итераций

\REDUCE - проводиться эксперимент по сужению (присутствует)/ не проводиться эксперимент по сужению (отсутствует)
	для флага \REDUCE значение выставлять не нужно, он либо указывается, либо нет

//===============================================================================================//

	по умолчанию: 12

\N - размер популяции
	по умолчанию: 50

\ITER - кол-во итераций

\PROBLEM_NAME - имя задачи из входного файла, которую надо решить (имя указывать в кавычках)

\FILE_PARETO - имя файла, в котором содержиться мн-во Парето (имя указывать в кавычках)

\TIME - время работы алгоритма над одной задаче на одном запуске (сек.)
	если флаг отсутствует, то работа по: (число итераций) OR (необновление архива/элиты)

\REDUCE - проводиться эксперимент по сужению (присутствует)/ не проводиться эксперимент по сужению (отсутствует)
	для флага \REDUCE значение выставлять не нужно, он либо указывается, либо нет

//===============================================================================================//
Останов алгоритма VNS_multi_start: \MAX_ELITE_VNS_MULTI
Останов формирования текущего архива: \ITER или \TIME или \MAX_ARCH_VNS_MULTI

\n - кол-во городов
	по умолчанию: 12

	по умолчанию: 50

	по умолчанию: 1

\FILE_PARETO - имя файла, в котором содержиться мн-во Парето (имя указывать в кавычках)
	по умолчанию: (отсутствует, сравнение не проводится)

\MAX_ARCH_VNS_MULTI - максимальное число итераций необновления архива
	по умолчанию: размер первоначального архива
	
\MAX_ELITE_VNS_MULTI - максимальное число итераций необновления элиты
	по умолчанию: 5
	
\TIME - время работы алгоритма над одной задаче на одном запуске (сек.)
	по умолчанию: работа по \ITER или \MAX_ARCH_VNS_MULTI

\REDUCE - проводиться эксперимент по сужению (присутствует)/ не проводиться эксперимент по сужению (отсутствует)
	для флага \REDUCE значение выставлять не нужно, он либо указывается, либо нет
